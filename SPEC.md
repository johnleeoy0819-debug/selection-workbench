## 2. 模块详细规范

### 模块 1：关键词工作台

**输入：** eRank CSV 文件（两种格式）
**输出：** 筛选后的候选关键词列表

#### 2.1.1 CSV 导入

**格式 1：关键词列表**
| 字段 | 类型 | 必填 |
|------|------|------|
| Keywords | string | ✅ |
| Average Searches | int | ✅ |
| Average Clicks | int | ✅ |
| CTR | int | ✅ |
| Competition | int | ✅ |
| KD | int | ✅ |
| Tag Occurrences | int | ❌ |
| Character Count | int | ❌ |
| Google Searches | int/string | ❌ |

**格式 2：Top Listings**
| 字段 | 类型 | 必填 |
|------|------|------|
| Keywords | string | ✅ |
| Age (Days) | int | ✅ |
| Views | int | ✅ |
| Daily Views | int | ✅ |
| Est. Sales | int | ✅ |
| Price | float | ✅ |
| Est. Revenue | float | ✅ |
| Hearts | int | ✅ |

#### 2.1.2 导入逻辑
- 自动检测 CSV 格式（关键词列表 vs Top Listings）
- 关键词列表 → `keywords` 表
- Top Listings → `competitor_listings` 表（关联到对应关键词）
- 返回导入统计：成功数/跳过数/错误数

#### 2.1.3 AI 关键词发散
- 输入：已选关键词 + 约束条件（品类、价格带、市场）
- 输出：10-15 个结构化关键词建议
- 包含：关键词、预估搜索量、竞争度、季节标记

#### 2.1.4 季节标记
- 自动标记每个关键词的旺季月份
- 时间状态：urgent（当月旺季）/ prepare（1-2月后）/ plan（3-6月后）/ early（6月+）/ evergreen（全年）

#### 2.1.5 人工初筛
- 表格展示所有关键词
- 支持：勾选保留/剔除、批量操作
- 筛选条件：KD 范围、搜索量范围、时间状态

**人工介入点：** 初筛（保留/剔除关键词）

### 模块 2：单品验证台

**输入：** 用户录入的产品数据 + 市场数据
**输出：** 完整的产品分析报告 + 三层验证结果

#### 2.2.1 产品身份录入
| 字段 | 类型 | 必填 |
|------|------|------|
| 产品名称 | string | ✅ |
| 关键词 | string（关联 keywords 表）| ✅ |
| 子类目 | enum | ✅ |
| 市场范围 | enum | ✅ |

#### 2.2.2 产品形态录入
| 字段 | 类型 | 必填 |
|------|------|------|
| 材质 | string | ✅ |
| 尺寸 | string | ✅ |
| 重量(g) | int | ✅ |
| 定制类型 | string | ❌ |
| 定制复杂度 | enum | ❌ |
| 生产周期(天) | int | ✅ |

#### 2.2.3 供应链录入
| 字段 | 类型 | 必填 |
|------|------|------|
| 供应商来源 | enum | ✅ |
| 供应商链接 | string | ❌ |
| 起订量(MOQ) | int | ✅ |
| 单价(CNY) | float | ✅ |
| 运费(CNY) | float | ✅ |
| 定制费(CNY) | float | ❌ |
| 包装费(USD) | float（默认 $0.8）| ✅ |

#### 2.2.4 自动成本计算
```
总 COG(USD) = (单价 + 运费 + 定制费) / 汇率 + 包装费
```

#### 2.2.5 自动定价计算
```
Etsy 平台费 = Listing费($0.20) + 交易额×6.5% + 支付费×3.5% + Offsite Ads(0% or 15%)
净利润 = 售价 - 总 COG - Etsy 平台费
净利率 = 净利润 / 售价 × 100%
```

#### 2.2.6 市场数据录入
| 字段 | 类型 | 必填 |
|------|------|------|
| 数据收集日期 | date | ✅ |
| Etsy 搜索结果数 | int | ✅ |
| 前10均价 | float | ✅ |
| 前10平均评论数 | int | ✅ |
| 前10卖家分布 | string | ❌ |

#### 2.2.7 竞品分析
- 录入前 5 个竞品 Listing 的详细数据
- 字段：排名、标题、价格、评论数、评分、上架天数

#### 2.2.8 三层验证
| 层级 | 验证内容 | 判定标准 |
|------|---------|---------|
| Layer 1 | 需求验证 | 搜索量 > 阈值，趋势向上 |
| Layer 2 | 竞争验证 | 有差异化空间，非红海 |
| Layer 3 | 利润验证 | 净利率 > 目标阈值 |

**人工介入点：** 所有数据录入（AI 无法获取 Etsy 前台和 1688 数据）

### 模块 3：AI 评分台

**输入：** 单品验证台的完整数据
**输出：** 评分报告 + 决策建议

#### 2.3.1 五维评分
| 维度 | 权重 | 评分依据 |
|------|------|---------|
| 需求 | 35% | 搜索量、点击率、趋势 |
| 利润 | 30% | 毛利率、净利、ROI |
| 竞争 | 25% | 竞品数量、差异化空间、前10集中度 |
| 节日 | 10% | 季节性强度、距离旺季时间 |

#### 2.3.2 综合评分
```
综合评分 = 需求分×0.35 + 利润分×0.30 + 竞争分×0.25 + 节日分×0.10
```

#### 2.3.3 决策建议
| 综合评分 | 建议 | 置信度 |
|---------|------|--------|
| 80-100 | 立即执行 | 高 |
| 60-79 | 观察测试 | 中 |
| <60 | 放弃 | 高 |

#### 2.3.4 预测记录
- 上架前预测：72h 曝光 / 7d 点击 / 7d 订单 / 30d 收入
- 置信度标记：高/中/低

**人工介入点：** 最终决策确认（AI 建议，用户拍板）

---

### 模块 4：选品池

**输入：** AI 评分台的结果
**输出：** 结构化的选品池

#### 2.4.1 入池
- 通过三层验证且综合评分 > 60
- 需用户确认

#### 2.4.2 状态机
```
待分析 → 分析中 → 待决策 → 已确认 → 已上架 → 观察中 → 已放弃 → 已归档
```

#### 2.4.3 排序筛选
- 排序：综合评分 / 入池时间 / 优先级 / 状态
- 筛选：子类目 / 价格带 / 状态 / 季节 / 评分区间

#### 2.4.4 批量操作
- 批量更新状态
- 批量导出 CSV

#### 2.4.5 每周推荐
- 自动推荐评分最高的 2-3 个品
- 需用户确认

**人工介入点：** 入池确认、状态更新、每周推荐确认

## 7. 风险与应对

| 风险 | 影响 | 应对 |
|------|------|------|
| AI 评分不准 | 高 | 预测→实际闭环校准，权重可调 |
| 汇率 API 失效 | 中 | 回退到 7.0，用户可手动更新 |
| 数据录入繁琐 | 中 | 表单优化，自动计算最大化 |
| 浏览器兼容性 | 低 | 现代浏览器即可 |
